TIPS
Modified a file? You need to git add it before you can commit it.
Staged the wrong file? git reset <file> unstages it — your edit is still there.
Uncommitted changes blocking a checkout? Stash them first with git stash.
Merge conflict? Find the <<<<<<< markers, edit them out, then add + commit.
Already shared a bad commit? Use git revert, not reset — it doesn't rewrite history.
git status is always safe to run. Use it whenever you're unsure.
git log --oneline gives you a quick map of recent commits.
Stuck on the current objective? Type :hint for a nudge.
